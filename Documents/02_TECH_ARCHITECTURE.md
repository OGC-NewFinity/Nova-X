# Technical Architecture

## System Overview
Nova-X operates as a WordPress plugin with a React-based frontend and PHP backend architecture.

## Core Components

### Frontend Stack
- **React** - UI component framework
- **Tailwind CSS** - Utility-first CSS framework
- **WordPress REST API** - Backend communication

### Backend Stack
- **PHP 8.0+** - Server-side logic
- **WordPress Core APIs** - Plugin integration
- **OpenAI API** - AI theme generation

### Infrastructure
- **Docker** - Development environment
- **WordPress 6.2+** - Platform requirement
- **MySQL 8.0** - Database

## Architecture Patterns
- Singleton pattern for core plugin class
- RESTful API design
- Component-based frontend architecture
- Modular PHP class structure
